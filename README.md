# AR Drawing

**AR Drawing** is a ad-free and pure offline AR drawing application.

## Features

*   **Lightweight:** The app size is under 10MB.
*   **Privacy-Friendly:** No needed network permission.
*   **All function**
    *   **Camera pause**
    *   **Opacity**
    *   **Size**
    *   **Flip (vertical & horizontal)**
    *   **Flashlight**
    *   **Keep screen on**
    *   **Lock**
    *   **Reset**
    *   **Open system camera**
